# FFmpegSimpleGUI

* FFmpegの簡易GUIです。 (Windows用)
* 動画の変換と連結に機能を絞っています。
* 動画の変換は基本的なパラメータに機能を絞っています。
* FFmpegは同梱しません。
* ffmpeg.exe にパスが通っていることが前提です。
* ffmpeg version n5.1.2で動作確認しています。古いバージョンのffmpegでは一部のオプションが正しく機能しないことがあります。
* Visual Studi 2019 / C# で作成したWPFアプリです。
